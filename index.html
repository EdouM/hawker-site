Hawker

Hawker redonne vie aux articles de journaux : en un clic, vous pouvez passer du texte à l'audio sur votre application mobile de journal favorite.

Présentation

Hawker permet de passer en un clic du texte à l'audio pour s'adapter aux nouvelles habitudes de lecture.

Technologie Text-to-Speech de pointe

Nous travaillons avec des chercheurs et utilisons du machine learning pour offrir une écoute agréable et unique.

API de synthèse de vocale

Notre API est intégrable à toute application mobile, pour vous permettre de lire votre contenu favori

Simple d'utilisation et transparent

Un clic suffit pour continuer la lecture la où vous vous êtes arrêtés.
